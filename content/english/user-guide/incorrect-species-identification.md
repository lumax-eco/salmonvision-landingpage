# 4. Incorrect Species Identification

The annotation tracked the fish accurately, however the species ID was incorrect. For example, a wrongly identified sockeye which is in fact a pink salmon.

![Wrongly identified fish](/images/user-guide/image029.gif)

**Action:** This can be adjusted by:

1. Locating the annotation in **Panel 7** (Annotations)
2. Once that individual is selected, locate and select the correct species ID in **Panel 3** (Class Overview)
3. This should change the detection to the correct species (e.g. pink salmon)
4. Once the video has been reviewed and submitted, the correct count for each species should be updated
