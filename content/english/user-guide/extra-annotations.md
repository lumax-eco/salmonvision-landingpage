# 5. Extra Annotations

At times multiple fish are detected when only one fish is present ("double annotation"). A common error by the AI model is detecting an additional fish — often this occurs in the last few frames, usually capturing the tail.

**Action:** Select the extra annotation (be sure you are not selecting the annotation that accurately tracks the individual) and delete it.
