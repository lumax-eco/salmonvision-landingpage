# Batch Upload Software

The batch upload software is used for offline sites to upload data from a detached hard drive to our cloud for analysis.

The software uses **rclone** under the hood, which will check the cloud before uploading — so it will only upload new or updated data to the cloud.

> If you plan to use the batch upload software, please contact the Salmon Vision team for support.
