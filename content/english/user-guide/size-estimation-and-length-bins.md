# Size Estimation & Length Bins

For projects using **SoundMetrics ARIS** sonar devices, fish sizes are automatically estimated using a pre-programmed length calibration equation that is built into the ARIS fish software.

However, each project may be interested in unique length bins for reporting purposes, since different species have unique length distributions. Accordingly, length bins for sonar projects can be configured into the backend of the Salmon Vision web app prior to syncing your data from cloud storage.

> When setting up your sonar data review project, please contact the Salmon Vision team for support with configuring the length bins.
