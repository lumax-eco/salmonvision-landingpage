User Guide / Dashboard Overview

Dashboard Overview

The SalmonVision dashboard is your central hub for monitoring salmon activity across all your projects.

Main Dashboard

After logging in, you’ll see:

  • Project Cards — Quick status overview for each project
  • Activity Feed — Recent detections and system events
  • Summary Statistics — Total counts across all active projects

Project View

Click on any project to see:

Live Feed Panel

The live view shows your camera feed with:

  • Real-time bounding boxes around detected salmon
  • Species labels and confidence scores
  • Frame rate and processing latency indicators

Count Panel

The count panel displays:

  • Running total — Fish counted in the current session
  • Species breakdown — Pie chart with counts per species
  • Hourly graph — Bar chart showing hourly passage rates

Timeline

The timeline view lets you:

  • Scrub through recorded footage
  • Jump to specific detection events
  • Filter by species, time of day, or confidence threshold

Filters and Controls

Use the toolbar to:

Control Description
Date Range Filter data by start and end date
Species Filter Show only selected species
Confidence Threshold Hide detections below a confidence level
Export Download data as CSV or JSON
