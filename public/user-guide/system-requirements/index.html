User Guide / System Requirements

System Requirements

Cloud Platform (Web App)

The SalmonVision web application runs in any modern browser:

Requirement Minimum
Browser Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
Internet 5 Mbps download
Display 1280×720 or higher

Edge Processing Unit

For on-site deployment with real-time detection:

Component Minimum Recommended
Processor NVIDIA Jetson Nano NVIDIA Jetson Orin Nano
RAM 4 GB 8 GB
Storage 32 GB microSD 128 GB NVMe
Power 5V 4A (USB-C) 12V DC
Network Wi-Fi or Ethernet Ethernet + LTE backup
Camera USB or RTSP IP camera GigE Vision camera

Supported Cameras

SalmonVision supports the following camera protocols:

  • RTSP — Most IP cameras
  • USB Video Class (UVC) — USB webcams and industrial cameras
  • GigE Vision — High-performance industrial cameras
  • MIPI CSI — Raspberry Pi and Jetson camera modules

Software Dependencies

For self-hosted deployments:

  • Docker 24.0+
  • Docker Compose 2.20+
  • Python 3.10+ (for custom model training)
  • CUDA 12.0+ (for GPU acceleration)
