User Guide / Camera Setup

Camera Setup

This guide covers how to configure camera sources for use with SalmonVision.

Adding a Camera

  1. Navigate to Project Settings → Camera Sources
  2. Click Add Camera
  3. Fill in the connection details:
name: "Upstream Weir Camera 1"
type: rtsp
url: rtsp://192.168.1.100:554/stream1
username: admin
password: ********
resolution: 1920x1080
fps: 15
  1. Click Test Connection to verify the feed
  2. Click Save

Camera Placement Best Practices

For optimal detection accuracy:

  • Underwater cameras — Position perpendicular to fish passage, 0.5–2m depth
  • Weir cameras — Mount overhead looking down at a 45–60° angle
  • Drone cameras — Fly at 10–30m altitude with nadir (straight-down) view

Image Quality Tips

Factor Recommendation
Resolution 1080p minimum, 4K preferred
Frame rate 15+ FPS for underwater, 30 FPS for aerial
Lighting IR illumination for low-light underwater conditions
Lens Wide-angle (120°+) for weir coverage

Managing Multiple Cameras

You can add multiple cameras per project. Each camera will:

  • Have its own live feed panel
  • Contribute to the aggregate count
  • Be independently configurable

Use camera groups to organize feeds by location or type.
